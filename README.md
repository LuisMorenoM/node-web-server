# Node + Express + Handlebars + Gulp + Sass web server

## Installation
```
npm install
```

## Usage
```
noder server.js
gulp watch
```

## Using
- [Node](https://nodejs.org)
- [Expressjs](https://expressjs.com)
- [Handlebarsjs](https://handlebarsjs.com/)
- [express-handlebars](https://github.com/ericf/express-handlebars)
